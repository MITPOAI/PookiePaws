# Code Of Conduct

## Our Standard

PookiePaws is an open-source project intended for practical, defensible engineering work. Participation in this repository requires respectful, professional behavior.

Expected behavior:

- discuss ideas directly without personal attacks
- critique code, docs, and decisions on their merits
- assume good faith until there is evidence otherwise
- keep discussions relevant to the repository and its goals
- respect privacy, security boundaries, and confidential information

Unacceptable behavior:

- harassment, abuse, intimidation, or discrimination
- deliberate disruption of discussions or review processes
- sharing private information without permission
- encouraging unsafe, deceptive, or malicious use of the project
- posting spam, slurs, or abusive content

## Enforcement

Maintainers may remove comments, close discussions, reject contributions, or restrict participation when behavior violates this code of conduct.

## Reporting

For conduct concerns, contact [support@mitpo.io](mailto:support@mitpo.io) with the subject line `PookiePaws Code of Conduct`.

Include:

- a short description of the incident
- links or screenshots when available
- the date and approximate time of the incident

## Project Focus

The repository exists to build a credible open-source foundation. Disagreement is acceptable. Low-signal hostility is not.
